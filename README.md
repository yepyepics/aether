# Aether

> A blazingly fast, minimalist yt-dlp GUI built with Tauri and SolidJS.

Aether is a desktop-first media downloading experience focused on speed, clarity, and a clean operational flow. It combines the performance of Rust and Tauri with a streamlined SolidJS interface to deliver a lightweight companion for `yt-dlp`.

## Features

- Smart Quality Cards for quick, intuitive format selection.
- SponsorBlock integration for cleaner downloads with skipped sponsored segments.
- Eco-mode for controlled bandwidth usage through speed limiting.
- Auto-updater support for seamless desktop releases.

## Tech Stack

- Rust
- Tauri
- SolidJS
- Tailwind CSS

## Development

Install dependencies:

```bash
npm install
```

Run the desktop app in development mode:

```bash
npm run tauri dev
```
